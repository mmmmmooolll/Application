Python Basics

Python is a versatile and powerful programming language known for its simplicity and readability. Let's start with some basic concepts:

Declaration

Python uses dynamic typing, meaning you don't need to declare variable types explicitly. Variables are created when assigned values.


      x = 5
      y = "Hello, World!"
      z = True

Functions

Functions in Python are defined using the def keyword. They can accept parameters and return values.

def greet(name):
          return "Hello, " + name
      
      result = greet("Alice")
      print(result)

Control Flow

Python supports common control flow constructs like if-else statements and loops.

if x > 10:
          print("x is greater than 10")
      else:
          print("x is less than or equal to 10")
      
      for i in range(5):
          print(i)

Operations on Sets

Common operations on sets include:

  • Adding elements
  • Removing elements
  • Union, intersection, and difference
  • Subset and superset checks

                
              # Creating a set
              my_set = {1, 2, 3}
                    
              # Adding elements
              my_set.add(4)
                    
              # Removing elements
              my_set.remove(3)

              intersection_set = set1.intersection(set2)
              difference_set = set1.difference(set2)

              is_subset = set1.issubset(set2)

Applications of Sets

Sets are useful for tasks involving:

  • Removing duplicates from a list
  • Checking for membership
  • Performing set operations in mathematics
